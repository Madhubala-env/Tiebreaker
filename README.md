# Decision Workshop AI

An AI-powered decision-making platform that helps users evaluate complex choices using weighted priorities, structured comparisons, SWOT analysis, and AI-generated strategic recommendations.

---

## 🚀 Run Locally

### Prerequisites

- Node.js (v18 or later recommended)

### Installation

1. Clone the repository

```bash
git clone <repository-url>
cd <repository-name>
```

2. Install dependencies

```bash
npm install
```

3. Create an environment file

Create a `.env.local` file in the root directory and add your Gemini API key.

```env
GEMINI_API_KEY=your_gemini_api_key
```

4. Start the development server

```bash
npm run dev
```

# 🧭 Core Functional Modules

## Decision Workshop

Describe your decision and compare up to **five custom options**.

Features include:

- Decision description
- Context input
- Personal priorities
- Multiple options comparison

---

## Quick Start Templates

Pre-built decision templates to get started instantly.

Examples include:

- 🚗 Electric Vehicle vs Hybrid
- 💼 Corporate Job vs Startup
- 🌍 Relocating to a New City

---

## Weighted Priority Balance Beam

Transform AI-generated pros and cons into adjustable priorities.

Features:

- Interactive importance sliders
- Scale from **0 (Ignore)** to **4 (Dealbreaker)**
- Real-time balance indicator updates
- Dynamic recommendation shifts based on user priorities

---

## Structured Criteria Matrix

Compare every option using context-aware evaluation criteria.

Examples include:

- Upfront Cost
- Long-Term Value
- Career Growth
- Risk
- Convenience
- Lifestyle Impact

Each criterion includes:

- Score out of **10**
- Adjustable importance weighting
- Real-time recalculation

---

## SWOT Analysis

Automatically generates a structured SWOT analysis covering:

- ✅ Strengths
- ⚠️ Weaknesses
- 📈 Opportunities
- 🚨 Threats

Presented using an organized Bento Grid layout.

---

## Strategic AI Verdict

AI provides a final recommendation including:

- Recommended option
- Confidence percentage
- Trade-off analysis
- Reasoning behind the recommendation
- Interactive next-step checklist

---

## History & Export

All decisions are stored locally in the browser.

Features include:

- Local history
- Restore previous analyses
- Save custom weight adjustments
- Export complete analysis as Markdown
